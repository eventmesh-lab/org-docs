# ADRs globales

Este documento indexa ADRs de la organización. Las ADRs locales a cada servicio se sincronizan y se pueden enlazar desde aquí.

Formato recomendado para ADRs en cada repo:
- docs/adr/NNNN-titulo.md
- metadata: Status, Date, Context, Decision, Consequences

Ejemplo de plantilla local de ADR: docs/adr/0001-template.md (ver repo de plantilla).
